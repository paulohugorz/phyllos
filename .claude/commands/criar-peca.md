# /criar-peca

Transforme uma ideia de peça em especificação inicial.

Referencias obrigatorias:
- `.claude/agents/references/motor-moldes-strategic-premises.md`
- `.claude/agents/references/fashion-os-platform-specialization.md`
- `.claude/agents/references/positioning-rationale-2026-06.md`

Fluxo:
design-tecnico → sourcing → custo-precificacao → modelagem → ficha-tecnica

Entrada:
```text
$ARGUMENTS
```

Entregue descrição técnica, categoria Fashion OS, tecido sugerido, Fit Engine preliminar, Fabric Engine preliminar, premissas de Pattern Engine, campos mínimos para o Motor de Moldes, riscos de modelagem, materiais, custo inicial e ficha técnica preliminar alinhada ao Kit PHYLLOS. Prompt de imagem só entra se ajudar a validar ou comunicar a peça; não é prioridade do MVP.
