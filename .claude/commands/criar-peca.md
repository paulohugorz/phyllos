# /criar-peca

Transforme uma ideia de peça em especificação inicial.

Referencias obrigatorias:
- `.claude/agents/references/fashion-os-platform-specialization.md`
- `.claude/agents/references/positioning-rationale-2026-06.md`

Fluxo:
design-tecnico → sourcing → custo-precificacao → modelagem → ficha-tecnica

Entrada:
```text
$ARGUMENTS
```

Entregue descrição técnica, categoria Fashion OS, tecido sugerido, Fit Engine preliminar, Fabric Engine preliminar, prompt de croqui, prompt de foto, materiais, custo inicial, premissas de Pattern Engine e ficha técnica preliminar alinhada ao Kit PHYLLOS.
