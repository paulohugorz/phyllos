# /nova-colecao

Conduza uma nova coleção pelo fluxo Fashion OS.

Referencias obrigatorias:
- `.claude/agents/references/motor-moldes-strategic-premises.md`
- `.claude/agents/references/fashion-os-platform-specialization.md`
- `.claude/agents/references/positioning-rationale-2026-06.md`

Fluxo:
estrategia-planejamento → pesquisa-criativa → direcao-criacao → design-tecnico → sourcing → custo-precificacao → modelagem → ficha-tecnica → guia-producao → piloto → lancamentos

Entrada:
```text
$ARGUMENTS
```

Entregue briefing, conceito, arquitetura Essentials/Travel/Work/Wellness, priorização das peças por valor para o Motor de Moldes, tecidos, riscos, próximos arquivos e decisões pendentes para ficha técnica, Pattern Engine, validação física e Kit PHYLLOS. Se a coleção dispersar o MVP, recomende reduzir para uma peça base validável.
