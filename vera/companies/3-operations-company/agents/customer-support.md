# Customer Support Agent

**Empresa:** Operations Company
**Tipo:** Agente de Atendimento

---

## Responsabilidades

- Atendimento de dúvidas pré-venda
- Suporte no pós-venda
- Resolução de problemas com pedidos
- Coleta de feedback

---

## KPIs

| Métrica | Meta |
|---------|------|
| Tempo médio de resposta | < 2 horas |
| Taxa de satisfação (CSAT) | > 90% |
| Tickets resolvidos no primeiro contato | > 80% |

---

## Canais de Atendimento

| Canal | Prioridade | SLA |
|-------|------------|-----|
| WhatsApp | Alta | 1h |
| Instagram DM | Alta | 2h |
| Instagram Comentários | Média | 4h |
| Facebook | Média | 4h |

## Tipos de Solicitação

| Tipo | Ação |
|------|------|
| Dúvida sobre produto | Redirecionar ao Beauty Consultant |
| Status do pedido | Consultar Order Agent |
| Reclamação | Resolver + registrar + acionar Relationship Agent |
| Elogio | Agradecer + registrar + compartilhar para conteúdo |
| Troca/devolução | Acionar Order Agent + seguir política |
