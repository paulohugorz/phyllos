# VERA OS v1.0
## Sistema Operacional — VERA | Cuidado & Beleza

> "Cuidar é um ato de inteligência."

VERA OS é a arquitetura operacional completa da VERA | Cuidado & Beleza — uma empresa de beleza e cuidado pessoal construída sobre dados, formulação de alta performance e experiência do cliente como vantagem competitiva.
